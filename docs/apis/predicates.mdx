## Decorator

### @predicate

Convert a comparison function into a Rue predicate with automatic metadata recording.

**Signature:**
```python
@predicate(
    func: Callable[..., bool | PredicateResult]
    | Callable[..., Awaitable[bool | PredicateResult]]
)
```

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `func` | `Callable` | Function that returns `bool` or `PredicateResult` and declares `actual` and `reference` as named positional parameters |

**Returns:** `Predicate` - A wrapped callable that returns `bool`

When a predicate returns `bool`, Rue builds the recorded `PredicateResult` from
the bound call. If the predicate declares `strict`, `confidence`, or `message`,
their recorded defaults come from the predicate signature itself.

**Example:**
```python
import re

from rue.predicates import predicate

@predicate
def mentions_price(actual: str, reference: str) -> bool:
    """Check if actual mentions the expected price."""
    prices = re.findall(r"\$[\d,]+(?:\.\d{2})?", actual)
    return reference in prices

@predicate
async def custom_semantic_check(actual: str, reference: str) -> bool:
    """Custom async predicate using LLM."""
    return await llm.evaluate(actual, reference)

@predicate
def within_tolerance(
    actual: float,
    reference: float,
    *,
    strict: bool = False,
    confidence: float = 0.8,
    message: str | None = "outside tolerance",
) -> bool:
    return abs(actual - reference) <= 0.1

def test_test():
    summary = "The product costs $299.99"
    assert mentions_price(summary, "$299.99")
    assert within_tolerance(10.05, 10.0)
```

## Result Classes

### PredicateResult

Metadata for a predicate evaluation. Rue records these objects internally for
assertion reporting and database persistence, and you can construct them
manually when you need explicit collection behavior.

**Attributes:**

| Name | Type | Description |
|------|------|-------------|
| `actual` | `str` | String representation of observed value |
| `reference` | `str` | String representation of expected value |
| `name` | `str` | Name of the predicate function |
| `strict` | `bool` | Strictness flag used in evaluation |
| `confidence` | `float` | Confidence score in `[0, 1]` |
| `value` | `bool` | Boolean outcome of the check |
| `message` | `str \| None` | Optional explanation or reasoning |

**Methods:**

| Method | Returns | Description |
|--------|---------|-------------|
| `record()` | `None` | Appends the result to the current collector (if any) |
| `__bool__()` | `bool` | Returns `value` without side effects |
| `__repr__()` | `str` | JSON representation (Pydantic `model_dump_json`) |

**Example:**
```python
from rue.context import predicate_results_collector
from rue.predicates import PredicateResult

def test_manual_collection():
    results = []

    with predicate_results_collector(results):
        result = PredicateResult(
            actual="Paris is the capital of France",
            reference="capital of France",
            name="has_facts",
            value=True,
            confidence=1.0,
            message="All facts present",
        )
        result.record()

    assert results == [result]
    assert bool(result) is True
```

## Built-in Semantic Predicates

All built-in predicates are async functions that return `bool`.

### has_conflicting_facts

Check if actual text contradicts facts in reference text.

**Signature:**
```python
async def has_conflicting_facts(
    actual: str,
    reference: str,
    strict: bool = False,
) -> bool
```

**Returns:** `bool` - `True` if contradictions were found

### has_unsupported_facts

Check if actual text contains facts not found in reference text (hallucinations).

**Signature:**
```python
async def has_unsupported_facts(
    actual: str,
    reference: str,
    strict: bool = False,
) -> bool
```

**Returns:** `bool` - `True` if unsupported facts were found

### has_facts

Check if actual text contains all facts from reference text.

**Signature:**
```python
async def has_facts(
    actual: str,
    reference: str,
    strict: bool = False,
) -> bool
```

**Returns:** `bool` - `True` if all required facts are present

### matches_facts

Check if actual and reference texts convey the same set of facts.

**Signature:**
```python
async def matches_facts(
    actual: str,
    reference: str,
    strict: bool = False,
) -> bool
```

**Returns:** `bool` - `True` if texts are factually equivalent

### has_topic

Check whether actual text is substantively about the topic described by reference.

**Signature:**
```python
async def has_topic(
    actual: str,
    reference: str,
    strict: bool = False,
) -> bool
```

**Returns:** `bool` - `True` if the topic is substantively present

### follows_policy

Check if actual text adheres to rules or policies in reference text.

**Signature:**
```python
async def follows_policy(
    actual: str,
    reference: str,
    strict: bool = False,
) -> bool
```

**Returns:** `bool` - `True` if compliant with policy

### matches_writing_style

Check if actual text has the same writing style as reference text.

**Signature:**
```python
async def matches_writing_style(
    actual: str,
    reference: str,
    strict: bool = False,
) -> bool
```

**Returns:** `bool` - `True` if style matches

### matches_writing_layout

Check if actual text follows the same structure as a reference template.

**Signature:**
```python
async def matches_writing_layout(
    actual: str,
    reference: str,
    strict: bool = False,
) -> bool
```

**Returns:** `bool` - `True` if layout matches

## Usage Patterns

### Combining Multiple Predicates

```python
from rue.predicates import (
    follows_policy,
    has_conflicting_facts,
    has_topic,
    has_unsupported_facts,
)

async def test_comprehensive_check(product_bot):
    product_data = "ThermoPro X500, $299, WiFi enabled, 2-year warranty"
    description = product_bot.generate(product_data)

    assert not await has_unsupported_facts(description, product_data)
    assert not await has_conflicting_facts(description, product_data)
    assert await has_topic(description, "smart home temperature monitoring")
    assert await follows_policy(description, "No superlatives like 'best'")
```

### Using Strict Mode

```python
from rue.predicates import has_facts

async def test_precision(report_bot):
    data = "Q3 revenue: $4.2M, margins: 23.5%"
    report = report_bot.summarize(data)

    assert await has_facts(report, "$4.2M revenue", strict=False)
    assert await has_facts(report, "23.5%", strict=True)
```

### Automatic Result Collection

When a predicate runs inside a Rue-transformed `assert`, Rue still records a
`PredicateResult` internally for reporting and database persistence. The
predicate call itself returns only `bool`.

```python
from rue.predicates import has_facts

async def test_detailed_analysis(bot):
    assert await has_facts("Text", "Reference")
    # Rue stores PredicateResult metadata for this assertion automatically.
```
