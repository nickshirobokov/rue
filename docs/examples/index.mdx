title: "Examples"
---

Run any example with Rue's runner from the repo root:

```bash
uv run rue test examples/rue_example_otel_spans.py --otel
```

## Example Files

| File | Focus |
| --- | --- |
| `rue_di_example.py` | Resource injection and test organization |
| `rue_example_captured_output.py` | Capturing stdout/stderr |
| `rue_example_case_groups.py` | `CaseGroup` iteration and grouped thresholds |
| `rue_example_cases.py` | `Case` iteration basics |
| `rue_example_concurrency.py` | Concurrent execution and shared resources |
| `rue_example_metrics.py` | Metrics, scopes, and aggregated assertions |
| `rue_example_outcomes.py` | `rue.skip()`, `rue.fail()`, and `rue.xfail()` |
| `rue_example_parametrize_and_tags.py` | Parametrization, tags, skip, xfail, and repeat |
| `rue_example_predicates.py` | Predicate assertions |
| `rue_example_run_inline.py` | `@rue.run_inline` and sync threading behavior |
| `rue_example_otel_trace.py` | Querying OpenTelemetry data with `otel_trace` |
| `rue_example_otel_spans.py` | SUT OpenTelemetry spans and custom span scopes |
